# Mason Crest - Premium Shopify Theme

A luxurious, dark-themed Shopify theme designed for premium health & fitness brands.

## Theme Overview

Mason Crest is a complete custom Shopify theme featuring:
- Premium dark aesthetic with graphite (#1A1A1A) and black (#000000) backgrounds
- Fully responsive design optimized for all devices
- Dynamic product and collection integration
- Complete e-commerce functionality
- 100% customizable through Shopify Theme Editor

## Brand Identity

### Colors
- **Primary Background:** Graphite #1A1A1A
- **Dark Accent:** Black #000000
- **Primary Text:** White #FFFFFF
- **Highlight Neutral:** Warm Off-White #F5F5F5

### Typography
- Modern sans-serif headings with wide letter spacing
- Clean, light body text with high readability
- Uppercase styling for premium feel

### Logo
- Minimalist "MASON CREST" wordmark
- Available in white, black, and warm off-white variants
- MC monogram favicon for browser tabs

### Homepage
1. **Hero Section** - Full-width hero with background image, headline, and CTA
2. **Brand Values** - 4-column grid showcasing premium quality, curated gear, peak performance, and fast shipping
3. **Featured Categories** - Dynamic category cards pulling from Shopify collections
4. **Products Carousel** - Horizontal scrolling product showcase
5. **Testimonials** - Customer reviews with star ratings

## Theme Customization

All sections are fully editable in Shopify's Theme Editor with settings for:

- **Text:** All headings, descriptions, button labels
- **Colors:** Backgrounds, text colors, buttons, accents
- **Images:** Logos, hero backgrounds, category images, product photos
- **Links:** All navigation and button destinations
- **Layout:** Spacing, alignment, visibility toggles
- **Products:** Dynamic product and collection selection

## Interactive Features

- Product image zoom on hover

## Installation

1. Upload theme files to Shopify
2. Navigate to Online Store > Themes
3. Activate Mason Crest theme
4. Customize via Theme Editor

## Setup Checklist

1. **Logo:** Upload your logo in Theme Settings > Logo
2. **Favicon:** Upload favicon in Theme Settings > Logo
3. **Brand Values:** Add icons to Brand Values section
4. **Categories:** Create collections and add to Featured Categories
5. **Products:** Add products to collections
6. **Homepage:** Customize hero section text and images
7. **Navigation:** Update menu text if needed
8. **Footer:** Add links to About and Support pages
9. **Legal Pages:** Create policy pages (Shopify auto-generates)
10. **Colors:** Adjust brand colors in Theme Settings if desired
